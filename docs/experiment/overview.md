# Experiment Overview

This section is only for reproducing the paper workflows.

## Scope

- Commands here are script-first.
- Pages in this section tell you which shell scripts to run and in what order.
- If you want the general API, data format, or training workflow, go back to [Quick Start](../getting_started/quick_start.md) or [Workflow](../workflow/data_format.md).

## Available Reproduction Paths

- [GFM-RAG Reproduction](gfm_rag.md)
- [G-reasoner Reproduction](g_reasoner.md)

## What Is Different From Workflow

- `Workflow` teaches the reusable repository interfaces.
- `Experiment` mirrors the experiment scripts in `scripts/`.
- `Experiment` does not expand script internals into a second general tutorial track.
